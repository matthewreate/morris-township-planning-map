Designed for sharing, discussion, and easy iteration on GitHub Pages.

About This Prototype

This map is a concept demo for a small Morris Township working group. It presents Neil’s TAC hotspot map in a cleaner format, adds contextual linework, and introduces a structure that can later support public input.

Hotspots come from the TAC source map. Destination markers are sample placeholders for discussion and should be replaced before a wider launch.
